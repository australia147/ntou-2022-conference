# 110 學年度教學實踐研究暨大學社會責任實踐計畫成果交流會

National Taiwan Ocean University · 2022.07.21 成果交流會官方網站

> 改自原 [Google Sites 版本](https://sites.google.com/email.ntou.edu.tw/2022-ntou-tpr-usr/)，採現代海洋風格重新設計。

## 結構

```
ntou_redesign/
├── index.html          首頁
├── agenda.html         議程
├── field-trip.html     場域踏查
├── transport.html      交通資訊與地圖
├── contact.html        聯繫我們
└── assets/
    ├── css/style.css
    └── js/main.js
```

## 開發

純 HTML / Tailwind CSS (CDN) / Lucide Icons，無需 build。
直接用瀏覽器打開 `index.html` 即可預覽。

## 部署

部署於 [Zeabur](https://zeabur.com)，自動從 `main` 分支部署。
